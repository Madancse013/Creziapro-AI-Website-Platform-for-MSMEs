## 🧠 Overview

Creziapro is a full-stack AI-powered website platform designed for MSMEs (Micro, Small & Medium Enterprises).

Instead of static websites, Creziapro enables businesses to:

* interact with users in real-time
* recommend services automatically
* generate instant pricing
* convert visitors into paying customers

---

## 🔥 Problem

Most MSMEs:

* don’t have modern websites
* struggle to convert visitors into leads
* manually handle pricing & inquiries
* lose potential customers due to slow response

---

## 💡 Solution

Creziapro solves this by providing:

* a modern, high-converting website
* an AI chatbot that understands user needs
* automated service recommendations
* instant pricing generation

---

## ✨ Key Features

* 🤖 **AI Chatbot**

  * Understands user requirements
  * Suggests services
  * Generates pricing instantly

* 🌐 **Dynamic Website**

  * Home, Services, Portfolio, Blog, Contact
  * Fully responsive and fast

* 💰 **Smart Pricing Engine**

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* TypeScript
* Tailwind CSS
* shadcn/ui
* Framer Motion

### Backend

* Node.js / Serverless APIs
* Hono

### State Management

* Zustand

### AI System

* Conversational AI chatbot
* Service recommendation logic
* Dynamic pricing engine

---

## 🏗️ Architecture (High-Level)

User → Frontend (React) → API Layer → AI Engine → Response

* Frontend handles UI & interactions
* Backend processes logic & requests
* AI system handles recommendations & pricing

I built the entire platform end-to-end, covering frontend, backend, AI logic, and system architecture.

- Designed and developed a fully responsive UI using React, TypeScript, and Tailwind CSS
